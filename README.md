1) Overview:

This project is to predict customer churn for a bank using a deep learning classification model. The goal was to build a beginner-friendly yet complete workflow to understand how Keras and TensorFlow operate when constructing, training, and improving neural networks.
I used the Kaggle interface, its inbuilt Keras interface along with Pandas, NumPy, and Matplotlib for data cleaning, preprocessing, and visualization. This project helped me learn how different layers work, how model performance changes with tuning, and how evaluation metrics behave.
